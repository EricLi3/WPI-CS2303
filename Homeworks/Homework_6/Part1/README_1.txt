Part 1 : This program has functions that allow for Adding two matrix objects, and two functions that lets us
add a float to a matrix object and also one that allows us to add a Matrix to a float. 

To run: 

Compile : Type "g++ Matrix.cpp run_matrix.cpp -o a.exe" into the terminal 

Run : From the Compile command, you should see a "a.exe" in your directory. Type "./a.exe" in the terminal to run.
